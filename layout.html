Ejemplos de Layout con Bootstrap 5

1. Containers

Un container es un contenedor básico en Bootstrap que nos ayuda a centrar y aplicar márgenes a nuestros contenidos. Bootstrap tiene dos tipos de containers: fixed y fluid.

2. Grid

El sistema de grid de Bootstrap utiliza una serie de filas y columnas para dividir el contenido en una estructura de 12 columnas. Puedes crear layouts complejos con diferentes combinaciones de columnas.
